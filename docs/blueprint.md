# **App Name**: Iligan RouteFinder

## Core Features:

- User Location Input: Allows users to input their current location and destination within Iligan City.
- Shortest-Path Calculation: Calculates the shortest path between the user's current location and destination using Dijkstra's algorithm based on LTFRB data.
- Route Visualization: Displays the calculated route on a map with clear visual indicators.
- Route Details Display: Presents detailed information about the calculated route, including estimated travel time and distance, as well as which route details apply for each segment.
- Admin Route Management: Provides an admin interface for adding, editing, updating, and deleting route data based on LTFRB information.
- LTFRB Data Import: Enables the admin to import data directly from LTFRB databases in standardized formats for quick addition and editing. Support for JSON formats

## Style Guidelines:

- Primary color: Light Blue (#ADD8E6) to evoke a sense of calm and trustworthiness, reflecting the reliability of the navigation app.
- Background color: Very Light Gray (#F0F0F0) provides a clean and neutral backdrop, ensuring legibility and focus on the map and route details.
- Accent color: Teal (#008080) for interactive elements and highlights, providing a distinct visual cue for important actions.
- Body and headline font: 'Inter', a grotesque-style sans-serif known for its neutral and readable design, which makes it perfect for both headlines and body text.
- Use clear and recognizable icons for map markers, route indicators, and points of interest.
- Maintain a clean and intuitive layout, with a prominent map display and clear route information panels.
- Subtle animations for route updates and transitions, enhancing user engagement without being distracting.
